# RF Athletics CRM

A comprehensive Customer Relationship Management system for tracking and analyzing athletics performance data.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

```
├── docs/              # 📖 All project documentation
├── src/               # 💻 Application source code
├── scripts/           # 🔧 Database utilities and scripts

└── public/            # 🎨 Static assets
```

## 🛠️ Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Database**: Supabase (PostgreSQL)
- **UI**: Tailwind CSS + shadcn/ui
- **State Management**: React Context

## 📦 Key Features

- Athlete performance tracking
- Event management and analytics
- Personal best (PB) tracking
- Benchmark comparisons
- Team management
