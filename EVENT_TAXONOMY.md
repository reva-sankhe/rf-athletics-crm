# Event Taxonomy - Standard Event Names

**Version:** 1.0  
**Date:** April 24, 2026  
**Format:** Gender's Distance/Event (Specification)

## Standard Format Rules

1. **Always include gender prefix:** `Men's`, `Women's`, or `Mixed`
2. **Use short distance format:** `100m` not `100 Metres`
3. **Capitalize event names:** `Long Jump` not `long jump`
4. **Keep weight/height specifications** as separate events when they exist
5. **For relays:** `Gender's 4x100m Relay`
6. **For race walks:** `Gender's 20km Race Walk`

## Usage Guidelines

### When to Add Gender Prefix
- **Always** include gender prefix for all events
- Use `Mixed` for relay events that are explicitly mixed-gender

### When to Include Specifications
- Keep specifications (e.g., `(6kg)`, `(91.4cm)`) when they appear in source data
- These represent different age categories or competition levels
- Standard events (without specs) are typically senior/elite level

### Distance Formats
- Use `m` for track distances: `100m`, `1500m`
- Use `km` for road/walk distances: `20km`, `35km`
- Full words for special: `Marathon`, `Half Marathon`, `Cross Country`

## Validation Rules

Event names MUST:
1. Start with `Men's`, `Women's`, or `Mixed`
2. Use exact capitalization as shown
3. Use `m` or `km` for distances (lowercase)
4. Match one of the 93 standard names exactly

## Notes

- **Women's Race Walk** (ID 93) is kept as a generic event for cases where distance is not specified
- Specifications in parentheses are kept to distinguish between age/level categories
- All relay events use the format `Gender's 4x###m Relay`
